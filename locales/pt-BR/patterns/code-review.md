# Code Review Assistido por IA

> IA é excelente em encontrar o que você deixou passar. Humanos são excelentes em julgar o que realmente importa.

## O Problema

Code review consome tempo e é inconsistente. Revisores deixam passar problemas de segurança, ignoram edge cases ou focam em estilo em vez de substância. Ao mesmo tempo, agentes de IA detectam esses problemas mecânicos rapidamente, mas não julgam corretamente lógica de negócio, decisões de UX ou aderência arquitetural.

O erro é tratar revisão por IA como substituta da revisão humana. O resultado é código com sintaxe certa, mas comportamento errado, ou tempo desperdiçado discutindo com a IA sobre decisões subjetivas.

## O Pattern

IA e revisores humanos têm forças complementares. Use ambos, para propósitos diferentes.
